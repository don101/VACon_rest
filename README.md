# VACon_rest
REST API combining Steam profile information and VAC Status

/friendslist/***STEAMID*** returns, for each friend of the Steam ID, a JSON list of objects that each contain:

<ul>
<li>Steam ID</li>
<li>Persona Name</li>
<li>Community URL</li>
<li>Avatar URL</li>
<li>VAC Status</li>
</ul>
